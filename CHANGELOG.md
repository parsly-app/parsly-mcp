# Changelog

All notable changes to `parsly-mcp` will be documented here.

## [0.0.1] - 2026-05-03

### Added
- Initial release
- `parsley_list_operations` — list all available browser automation operations
- `parsley_run_operation` — run an operation and save results to JSON
- `parsley_get_status` — check Chrome extension connection status
- `parsley_cancel_run` — cancel a running operation
- Stdio transport for Claude Desktop, Cursor, and ChatGPT
- HTTP transport (`parsly-mcp-http`) for multi-client setups
- WebSocket bridge to Parsly Chrome extension
